#  Sapphire Wallet

Sapphire Wallet is a POC of Account Abstraction using Argent's wallet contract.

## Prerequisites

* [Node.js](https://nodejs.org/en/) (18.18.2)
* [Yarn](https://yarnpkg.com/) (1.22.19)

## Install

```bash
yarn install
```

## Run

```bash
yarn start
```
